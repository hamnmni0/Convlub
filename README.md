# ConvLub Archive Website

This version of ConvLub is a storytelling website for past events.

It now focuses on:

- past event documentation
- a featured first edition called `Conversation Club`
- gallery-ready sections for event photos
- short reflections on how each event went
- light mode and dark mode toggle

- [script.js](C:\Users\ahmed\OneDrive\Documents\New project\script.js)

## How to update the site

When you finish a new event:

1. Add a new event card in `index.html`
2. Add a matching event story section
3. Update the text with a short reflection
4. Replace the gallery placeholders with real event photos

## Current note about the photos

The gallery structure is ready, but the uploaded chat images are not stored as actual project files inside this folder. To fully show them on the site, place the image files in the project folder and swap the gallery placeholders for real `<img>` tags.
